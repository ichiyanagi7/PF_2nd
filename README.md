# Record

## サイト概要
このサイトでは、自分だけの音楽日記をつけることができます。
一日一曲、聞いていた音楽とその日の出来事を一言だけ記録できます。
記録したものは、他のユーザーに公開されることはありません。
また、月毎に登録した曲が一覧で表示されます。

### サイトテーマ
音楽で思い出を振り返ろう

### ターゲットユーザ
- 音楽が好きで、毎日色々な曲を聞いているユーザー

### 主な利用シーン
- 就寝前のちょっとした時間
